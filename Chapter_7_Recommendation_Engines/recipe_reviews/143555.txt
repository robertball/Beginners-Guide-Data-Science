I make this regularly, this is "WOW!" if done right.  A helpful summary of tips: 
First and foremost definitely use 1/3 cup of reviewer Debbie's homemade onion soup mix (a batch includes: 3/4 c minced onion, 1/3 no MSG beef bouillon powder (low salt also if possible, I use Epicure's), a heaping Tbsp onion powder, 1/4 tsp celery salt, 1/4 tsp sugar; the unused remainder stores well in an airtight container). I use this for homemade veggie dips all the time too!
Secondly, at least double the sesame oil, ginger, garlic, sugar in the dressing, to your tastes. Also use orange juice concentrate instead of OJ itself. This all gives the dressing a more intense flavour, which is needed. The original recipe is a bit bland as written. You can use red pepper flakes if you would like a kick to it.
This salad tastes better on day 2. 
Refrigerate  at least for 4-5 hours before serving to allow flavours to meld, but don't add the leafy greens until just before serving so they aren't wilted.
I try to use whole grain pasta if I can find WG bowtie for better nutrition. I usually serve this as a side dish so I don't  include chicken usually. Sometimes I add other veggies like broccoli instead of spinach. 
I can only give 4 stars as written but this is so worth making!!!
